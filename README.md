# deep-agents

This repository contains examples for trying out the [LangChain deepagents](https://github.com/langchain-ai/deepagents) framework.

## About LangChain Deepagents

Deepagents is an agent harness built on LangChain and LangGraph. Deep agents are equipped with:
- A planning tool
- A filesystem backend
- The ability to spawn subagents

This makes them well-equipped to handle complex agentic tasks.
